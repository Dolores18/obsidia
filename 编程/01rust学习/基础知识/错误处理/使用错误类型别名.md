使用类型别名之后，写函数的时候就不需要每次都返回Rresult(T, E)了
```rust
use std::io;

// 定义自己的错误类型
enum MyError {
    IoError(io::Error),
    CustomError(String),
}

// 创建类型别名
type Result<T> = std::result::Result<T, MyError>;

// 使用新的 Result 类型
fn some_function() -> Result<i32> {
    // 函数实现...
    Ok(42)
}

fn main() {
    match some_function() {
        Ok(value) => println!("Success: {}", value),
        Err(e) => println!("Error: {:?}", e),
    }
}
```
