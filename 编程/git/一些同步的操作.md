

# 修改之后提交云端
	git add .
	git commit -m ' '
	git  push

# 从云端拉取
	git pull obsidia master
	git add .
	git commit -m ""
	git restore .  //path/to/your/file
# 同步操作
	git add .
	git commit -m "Android Sync $(date)"
	git pull --no--edit 
	git push 

# 同步过程中遇到冲突

1. 打开冲突的文件，进行编辑，把那些head什么的标记都给删除。
2. 然后直接 add . , 如果不怕麻烦的话就可以add特定的文件夹
3. 然后就是 git rebase --continue
4. 接下来再次pull ,如果有冲突就继续修改
5. push
