1.基本操作
```
# 打开文件
vim filename.txt

```
2.模式切换
```
# 进入插入模式
i   # 在光标前插入
a   # 在光标后插入
o   # 在当前行下面新开一行并插入

# 退出插入模式
<Esc>  # 按下 Esc 键

```
3.保存和推出
```
:w       # 保存文件
:q       # 退出 Vim
:wq      # 保存并退出
:x       # 保存并退出
:q!      # 强制退出，不保存
:wq!     # 强制保存并退出

```
4.光标移动
```
h       # 向左移动光标
j       # 向下移动光标
k       # 向上移动光标
l       # 向右移动光标

0       # 移动到行首
$       # 移动到行尾
gg      # 移动到文件开头
G       # 移动到文件结尾
:n      # 移动到第 n 行，如 :10 移动到第 10 行

```
5.编辑操作
```
x       # 删除光标所在字符
dd      # 删除当前行
yy      # 复制当前行
p       # 粘贴复制或剪切的内容到光标后
u       # 撤销操作
Ctrl + r # 重做操作

```
6.搜索与替换
```
/word   # 向下搜索 'word'
?word   # 向上搜索 'word'
n       # 跳到下一个匹配项
N       # 跳到上一个匹配项

:%s/old/new/g      # 全文替换 'old' 为 'new'
:%s/old/new/gc     # 全文替换 'old' 为 'new'，并逐一确认

```
7.其他实用命令
```
:help keyword  # 打开 Vim 帮助并搜索 'keyword'
:split file    # 水平分割窗口并打开 'file'
:vsplit file   # 垂直分割窗口并打开 'file'
:tabnew file   # 新建标签页并打开 'file'
:buffers       # 列出所有缓冲区
:bn            # 切换到下一个缓冲区
:bp            # 切换到上一个缓冲区

```
8.可视模式
```
v       # 进入可视模式，用于选择文本
V       # 进入行可视模式，选择整行
<Esc>   # 退出可视模式


```
