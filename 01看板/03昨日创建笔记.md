```dataview 
table file.ctime
where file.cday = date(yesterday) 
sort file.ctime asc limit 100
```



```dataviewjs 
dv.table(
  ["笔记", "创建时间"], // 表头
  dv.pages()
    .filter(n => 
      n.file.ctime >= dv.date('yesterday') &&
      n.file.ctime < dv.date('today'))
    .map(n => [n.file.link, n.file.ctime]) // 映射到文件链接和修改时间
)
```
